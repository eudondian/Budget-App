# ğŸ“– BUDGET APP <a name="about-project"></a>


**BUDGET APP**  
This Ruby on Rails capstone project involves developing a mobile web application for budget management. The application allows users to maintain a record of transactions categorized by type. This enables users to track their expenditure, providing insights into their spending patterns and categories.

### Key Features <a name="key-features"></a>

- **User**: 
   - User Registration and Authentication
   
        The application offers user registration and secure login functionality, safeguarding financial information and customizing the user experience.

   - Efficient Transaction Tracking

      Users can effortlessly record new transactions within the application, including the transaction amount and its association with specific categories. This capability ensures accurate and comprehensive expense tracking.
   - Visualizes and analyzes expenses for each category.

        By presenting the total expenditures within each category, the application delivers valuable insights into users' spending behavior. This feature empowers users to recognize significant spending areas and make well-informed financial choices.

## ğŸ’» Getting Started <a name="getting-started"></a>


To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

```sh
 install ruby
 ruby --version
 install sqlite3
 sqlite3 --version
 gem install rails 
 rails --version
```


### Setup

Clone this repository to your desired folder:


```sh
  cd my-folder
  git clone https://github.com/eudondian/Budget-App.git
```
### Install

Install this project with:

```sh
  cd budget-app
  bundle install
```
### Usage

To run the project, execute the following command:

```sh
  rails server
```
